# zerasfood

Aplicación móvil para registrar y analizar los ingresos y gastos de un pequeño restaurante de comidas rápidas que opera los fines de semana y días festivos.

**Descripción**

Zera'sFood permite llevar un control sencillo de la economía del negocio, registrando tanto las ventas como los gastos diarios (insumos, servicios, otros). La app organiza la información por fechas y categorías, facilitando la revisión del historial financiero y ayudando a identificar tendencias de ingresos y costos.

**Entre sus objetivos están:**

Centralizar el registro de ingresos y gastos del restaurante.

Ofrecer una vista rápida del balance de cada jornada.

Permitir analizar el comportamiento del negocio en fines de semana y festivos.

**Tecnologías**

El proyecto está desarrollado con Flutter y utiliza una base de datos local (por ejemplo, sqflite) para almacenar la información directamente en el dispositivo, lo que permite usar la app sin conexión.

**Uso básico**

1.Clonar o descargar el repositorio.

2.Ejecutar flutter pub get para instalar dependencias.

3.Con un dispositivo o emulador configurado, ejecutar:

```
flutter run
```
4.Registrar ingresos y gastos desde la interfaz de la aplicación, seleccionando la fecha y categoría correspondiente.

Zera'sFood está pensado como una herramienta práctica y ligera para dueños de pequeños negocios que quieren controlar mejor sus números sin recurrir a sistemas complejos.
